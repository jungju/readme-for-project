# Smart README.md

## usage
```
readme-for-projects -token={{GITHUB_TOKEN}} -user={{GITHUB_USER}} -repo={{GITHUB_REPO}} -file=./README.md
```

### With token of env

```
GITHUB_TOKEN={{GITHUB_TOKEN}} readme-for-projects -user={{GITHUB_USER}} -repo={{GITHUB_REPO}} -file=./README.md
```
